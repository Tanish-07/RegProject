Register App

During today’s offline interaction, we were asked to develop a User Registration Application that allows users to create an account and access the system through authentication.

The application provides functionality where users can register themselves and log in to the system to access their dashboard. The system also includes an Admin login, where the admin credentials are predefined in the system and only the admin can access administrative features.

The main objective of this project is to build a simple and structured authentication system using modern web technologies while ensuring basic access control between Admin and Normal Users.

Key Functional Requirements

User Registration (Sign Up) User Login Authentication Admin Login (Predefined Credentials) Role-based Access (Admin & Normal User) Angular modules & services
